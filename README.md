# Smart Locator Generator + HTML Validator

A React-based tool that generates Selenium-ready locators from HTML and validates HTML structure.

## Features

- Generate ID, CSS, XPath, Name, Class, and Tag locators
- Locator Strength Indicator
- Smart Label Detection
- Element Type Detection
- Search & Filter Locators
- Copy Individual Locator
- Copy All Locators
- Export CSV
- HTML Validation Report
- Responsive UI

## Tech Stack

- React
- JavaScript
- Tailwind CSS
- Vite

## Installation

```bash
npm install
npm run dev
